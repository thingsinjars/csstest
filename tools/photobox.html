Grunt-Photobox

This project's purpose is to make the check for changed or broken layout easier. If you use Grunt you can easily set it up inside of your Gruntfile.

It will take pictures of your site in defined resolutions and give you the oppurtunity to compare it with the pictures of the previous run via a generated index.html file.

For comparison there are two modes included. The first one works without any dependencies and will generate an index page that included some JavaScript functionality to overlay the previous and the current picture. A tutorial of this mode can be found here.

The second mode make usage of ImageMagick and will generate nice diff images for easier and quicker comparison.
